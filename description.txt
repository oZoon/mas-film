///////////////////////////////////////////////////////////////////////////////
СПИСОК ФИЛЬМОВ
///////////////////////////////////////////////////////////////////////////////

базовый запрос:
getAllFilms

параметры (необязательно):
filterActor
filterGenre
sort

пример
GET /?action=getAllFilms&filterActor=abc&filterGenre=bcd&sort=ASC
ОТВЕТ:
[
    {
        "filmID": "1",
        "faimName": "ABC"
    },
    {
        "filmID": "2",
        "faimName": "BCD"
    },
    {
        "filmID": "3",
        "faimName": "CDE"
    },
]
